# Hardware

This folder is intended for hardware-related documentation and design assets for the wearable sensing module.

Future files may include:

- ESP32 wiring notes.
- Sensor connection diagrams.
- Bill of materials.
- Wearable enclosure design notes.
- Pulse sensor, water immersion sensor, and MPU6050 interfacing details.

Hardware files should remain clear, reproducible, and safe for public sharing.
