# Diagrams

This folder contains Mermaid diagrams and flowcharts for the Drowning Detection and Rescue System.

The diagrams are grouped by technical status:

| File | Status | Purpose |
| --- | --- | --- |
| [overall_system_architecture.md](overall_system_architecture.md) | Implemented | Shows the current wearable sensor, ESP32, fog, and alert architecture. |
| [event_triggered_workflow.md](event_triggered_workflow.md) | Implemented | Shows the current threshold-based drowning detection workflow. |
| [esp_now_communication.md](esp_now_communication.md) | Implemented | Shows the current ESP-NOW peer-to-peer and control-station communication flow. |
| [sensor_interaction_flow.md](sensor_interaction_flow.md) | Implemented | Shows how current sensor readings interact with ESP32 and detection logic. |
| [future_ai_architecture.md](future_ai_architecture.md) | Future work | Shows a possible AI-assisted extension. |
| [shoreline_deployment_concept.md](shoreline_deployment_concept.md) | Conceptual | Shows a possible shoreline-scale architecture. |
| [research_roadmap.md](research_roadmap.md) | Future work | Shows the research roadmap from current implementation to future study areas. |

Conceptual and future-work diagrams are not implemented system claims.
