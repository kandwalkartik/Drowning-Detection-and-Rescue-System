# Communication

This folder is intended for communication protocols and message formats used between wearable nodes, fog nodes, dashboards, and future mesh-network components.

Future files may include:

- ESP32 wireless communication notes.
- MQTT topic and payload definitions.
- Sensor message schemas.
- Gateway communication documentation.
- Future ESP32 mesh networking experiments.

The current project focuses on transmitting sensor readings for fog-layer evaluation. Future work may expand this area to support shoreline-scale communication through mesh networking and MQTT-based event delivery.
