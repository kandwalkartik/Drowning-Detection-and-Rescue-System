# Fog Layer

This folder is intended for local fog-computing logic that processes sensor data close to the aquatic environment.

Future files may include:

- Fog-node processing scripts.
- Threshold and rule evaluation logic.
- Sensor-data validation.
- Alert-triggering integration.
- Local logging or event-summary generation.
- Future edge AI inference experiments.

The current architecture uses fog computing to reduce latency and avoid relying only on cloud processing. Detection is currently described as threshold-based and rule-based.
