# Sensor Processing

This folder is intended for processing raw readings from the wearable sensors before decision logic is applied.

Future files may include:

- Signal filtering utilities.
- Sensor calibration notes.
- Immersion-duration calculation.
- IMU motion-window processing.
- Feature extraction for threshold-based detection.
- Feature extraction for future AI experiments.

Current processing is expected to support rule-based detection. Future work may extend this folder with reusable feature-preparation code for machine learning, but no ML model is currently claimed as implemented.
